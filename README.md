## Project: Sunday
A project management tool

## Start the project locally
To start the project locally, you should have docker installed in your machine

### Copy and edit your env files
```cp env.example .env```

**NAMESPACE** : the namespace of your project (ex: sunday)

**APP_ENV** : the environment you're supposed to run at (development / staging / production)

### Start the project

**Command to start**: ```make start-f```

To show makefile's help, just type the command : ```make``` in the root of the project
